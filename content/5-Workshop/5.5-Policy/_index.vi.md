#### Mục tiêu

Trong phần này, chúng ta sẽ triển khai giao diện web cho hệ thống:
- Xây dựng **Next.js** dashboard hiển thị dữ liệu thời tiết thời gian thực
- Deploy lên **AWS Amplify** với CI/CD từ GitHub
- Cấu hình **Amazon Cognito** xác thực người dùng
- Kết nối dashboard với **API Gateway + Lambda**

{{% notice info "Information" %}}
Nội dung chi tiết từng bước thực hiện sẽ được cập nhật khi triển khai thực tế.
